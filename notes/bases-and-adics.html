Bases & Adics

A fast, visual feel for place value across bases and how that flips into \(p\)-adic thinking.

Chapter A · Why base 10 feels natural, and why “2+2” can look like “10”

We grow up in base 10: ten symbols \(0,1,\dots,9\), and each place is a power of \(10\). For example, \(745\) means \(7\cdot 10^2 + 4\cdot 10^1 + 5\cdot 10^0\).

The value “two plus two” is always the value four. What changes across bases is how we write four:

  • Base 10: \(4\).
  • Base 4: \(10\) since \(1\cdot 4 + 0 = 4\).
  • Base 3: \(11\) since \(1\cdot 3 + 1 = 4\).
  • Base 2: \(100\) since \(1\cdot 2^2 = 4\).

So you can see “\(2+2=10\)” in base 4, or “\(2+2=100\)” if you choose base 2 for the result. (In base 2 the digit “2” isn’t allowed, so fully base-2 you’d write \(10+10=100\).)

Rule in base \(b\ge 2\): allowed digits are \(0,1,\dots,b-1\). A string \(d_k d_{k-1}\dots d_1 d_0\)_\(b\) means \(\;d_k b^k + d_{k-1} b^{k-1} + \dots + d_1 b + d_0\).

Quick arithmetic in other bases

Carrying is the same, but you carry when you hit the base.

Base 5 example: \(24_5 + 33_5\).

Rightmost: \(4+3=7=12_5\) → write \(2\), carry \(1\).
Next: \(2+3+1=6=11_5\) → write \(1\), carry \(1\).
Final carry gives \(112_5\). Check: \(24_5=14\), \(33_5=18\), sum \(32\), and \(112_5=32\). ✔︎

Multiplication is the same place-value algorithm with the base’s carry threshold.

Chapter B · Converting between bases (integers & fractions)

Integer → base \(b\) (division remainders)

Repeat divide by \(b\). Collect remainders bottom to top.

  • Example: convert \(117\) to base \(7\).
    \(117 = 7\cdot 16 + \color{#9cf}{5}\) ⟶ rem \(5\)
    \(16 = 7\cdot 2 + \color{#9cf}{2}\) ⟶ rem \(2\)
    \(2 = 7\cdot 0 + \color{#9cf}{2}\) ⟶ rem \(2\)
    Read upward: \(117 = \mathbf{225_7}\).

Base \(b\) → base 10 (expand by powers)

  • \(1402_7 = 1\cdot 7^3 + 4\cdot 7^2 + 0\cdot 7 + 2 = 343 + 196 + 0 + 2 = \mathbf{541}\).

Fractional part → base \(b\) (multiply by \(b\))

Multiply by \(b\); each whole part is the next digit.

  • \(0.375\) to base \(2\): \(0.375\cdot 2=0.75\) (digit \(0\)); \(0.75\cdot 2=1.5\) (digit \(1\), keep \(0.5\)); \(0.5\cdot 2=1.0\) (digit \(1\)). So \(0.375=0.\!011_2\).
  • \(0.2\) to base \(3\): digits \(0,1,2,1,\overline{21}\). So \(0.2=0.\!0121\overline{21}_3\).

Two quick base-to-base shortcuts

  • If one base is a power of the other, group digits (e.g., base 2 → base 8 in blocks of 3 bits).
  • For powers of ten, shifting the decimal point is the familiar trick.

Chapter C · From place value to adic thinking

Flip the mindset: measure closeness by divisibility rather than difference size. Two integers are “close” if their difference has many factors of a chosen base.

“Closeness” by powers

  • If \(a\equiv b \pmod{10}\), they match in the last decimal digit.
  • If \(a\equiv b \pmod{100}\), they match in the last two digits.
  • Matching mod \(10^k\) for big \(k\) means many matching tail digits.

This leads to 10-adics: left-infinite decimals \(\cdots d_3 d_2 d_1 d_0\) that stabilize modulo \(10^k\). Arithmetic is ordinary carrying, to the left, and respects all mod-\(10^k\) tails.

Famous identity: \(\cdots 9999 + 1 = 0\) (10-adically). Each truncation \(9,99,999,\dots\) is \(-1\) mod \(10^k\), so the limit acts like \(-1\).

Geometric series also converge 10-adically: \(\;1 + 10 + 10^2 + \cdots = \dfrac{1}{1-10} = -\dfrac{1}{9}\), since powers of 10 get 10-adically small.

Why switch to \(p\)-adics

Using a prime \(p\) avoids zero divisors. \(\mathbb Z_p\) has clean arithmetic, and \(\mathbb Q_p\) is a field—like \(\mathbb R\) but with a different absolute value.

Chapter D · Constructing \(p\)-adics & doing math in them

The \(p\)-adic valuation and absolute value

For nonzero \(n\), \(v_p(n)\) is the exponent of \(p\) in \(n\). Examples: \(v_5(250)=3\), \(v_3(27)=3\), \(v_2(12)=2\).

\(|n|_p = p^{-v_p(n)}\) (and \(|0|_p=0\)). So \(|p|_p=1/p\). Distance: \(d_p(a,b)=|a-b|_p\).

Ultrametric: \(|x+y|_p \le \max(|x|_p,|y|_p)\). Triangles are isosceles or equilateral.

What \(\mathbb Z_p\) and \(\mathbb Q_p\) are

  • \(\mathbb Z_p\): all left-infinite base-\(p\) strings \(\cdots d_3 d_2 d_1 d_0\) with \(d_i\in\{0,\dots,p-1\}\), agreeing mod \(p^k\) for all \(k\).
  • \(\mathbb Q_p\): allow division by numbers not divisible by \(p\); equivalently, complete \(\mathbb Q\) under \(|\cdot|_p\).

Digit expansions & arithmetic

Every \(x\in\mathbb Z_p\) has \(x=a_0+a_1p+a_2p^2+\cdots\), \(a_i\in\{0,\dots,p-1\}\). Add/multiply with left carries.

  • \(-1\) in \(\mathbb Z_p\) is \(\cdots (p-1)(p-1)(p-1)\).
  • Geometric series in \(\mathbb Z_5\): \(1+5+25+\cdots=\dfrac{1}{1-5}=-\dfrac{1}{4}\).

Inverses when \(p\nmid m\)

If \(p\nmid m\), then \(m\) is invertible in \(\mathbb Z_p\). Lift modular inverses:

Example: \(1/3\) in \(\mathbb Z_5\). Since \(3\cdot 2\equiv 1\pmod 5\), start with \(2\). Lift to mod \(25\) via \(2+5t\): \(3(2+5t)\equiv 1\Rightarrow 15t\equiv 20\Rightarrow t\equiv 3\). So \(a\equiv 17\pmod{25}\) and the base-5 digits emerge by continuing.

Hensel lifting for roots

If \(f(x)\equiv 0\pmod p\) with \(f'(x_0)\not\equiv 0\), you can lift to \(p^k\) for all \(k\) (hence to \(\mathbb Z_p\)).

  • Example: \(x^2=2\) in \(\mathbb Z_7\). Mod \(7\), \(3^2\equiv2\). One Newton step gives \(x_1\equiv 10\pmod{49}\); further steps give the full \(7\)-adic root.

Convergence & series

\(\sum a_n\) converges in \(\mathbb Q_p\) if \(a_n\to0\) in \(|\cdot|_p\) (often each term gains a factor of \(p\)). Power series like \(\exp,\log,(1+x)^\alpha\) behave cleanly for \(p\)-adically small \(x\).

Chapter E · Uses, problems, and a clean end at \(p\)-adics

What \(p\)-adics are good for

  • Solving congruences systematically (Hensel lifting).
  • Local–global methods (e.g., Hasse–Minkowski for quadratic forms).
  • Prime-focused computation with calculus-like tools in number theory.
  • Efficient work mod \(p^k\) (roots, factoring, coding/crypto contexts).
  • Number-theoretic functions (e.g., \(p\)-adic zeta and \(L\)-functions).

Common pitfalls

  • At \(p\), sizes invert: powers of \(p\) get smaller.
  • Left-infinite tails are normal; repeating \(p\!-\!1\) is \(-1\).
  • \(\mathbb Q_p\) is a field only when \(p\) is prime.

Practice problems

1) Convert \(300\) to base \(7\).
\(300 = 6\cdot 49 + 6\cdot 7 + 6 = \mathbf{666_7}\).
2) Convert \(0.2_{10}\) to base \(2\) (≈ 8 places).
\(0.2 \approx 0.\!0011\,0011_2\) (repeating \(0011\)).
3) Add in base \(6\): \(452_6 + 345_6\).
\(452_6 + 345_6 = \mathbf{1141_6}\).
4) Show \(\cdots 4444_5 = -1\) in \(\mathbb Z_5\).
Truncations \(4,44_5,444_5,\dots\) are \(-1 \bmod 5^k\); the 5-adic limit is \(-1\).
5) First two base-5 digits of \(1/3\) in \(\mathbb Z_5\).
\(3^{-1}\equiv 2\pmod 5\Rightarrow a_0=2\). Lift to \(25\): \(a_0+a_1\cdot 5\equiv 17\Rightarrow a_1=3\).
6) Lift a root of \(x^2\equiv 2\pmod 7\) to a root \(\bmod 49\).
Start \(x_0=3\). One Newton/Hensel step gives \(x_1\equiv 10\pmod{49}\).
A tidy picture. Bases change notation, not value. Adic closeness = many shared tail digits. With prime \(p\), \(\mathbb Q_p\) is a clean field with its own geometry of size—powerful for lifting solutions and doing analysis in number theory.
